# 🤖 Nova - Rule-Based AI Chatbot

**Project 1 | DecodeLabs AI Internship | Batch 2026**

A deterministic, rule-based chatbot built with pure Python. No machine learning, no neural networks — just control flow, hash maps, and logic. This is the foundation of AI engineering: understanding how to teach a machine through explicit instructions before diving into probabilistic systems.

---

## 🧠 Concept

| System 2 (Deterministic) | System 1 (Probabilistic) |
|---------------------------|---------------------------|
| Nova — this project | LLMs like GPT |
| Rule-based, traceable | Statistical, black-box |
| Zero hallucination risk | Requires guardrails |

This project demonstrates the **white-box** approach: Input → Logic → Output. Every response is traceable, safe, and 100% hard-coded.

---

## ⚙️ Features

- ✅ Infinite conversation loop with clean exit strategy
- ✅ Input sanitization (case-insensitive, whitespace handling, empty input safety)
- ✅ Dictionary-based knowledge base with 20+ intents
- ✅ O(1) constant-time lookup — no if-elif ladders
- ✅ User name memory across conversation turns
- ✅ Randomized joke bank for variety
- ✅ Friendly, varied fallback responses for unknown inputs
- ✅ Professional terminal interface
